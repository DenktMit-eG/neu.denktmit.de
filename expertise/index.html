CRUD Framework
Das CRUD Framework ist eine Library, um unnötigen Boilerplate Code zu vermeiden. Ist für eine Entität keine Logik erfolderlich, so muss man lediglich ein Model mit Annotationen erstellen, um alle CRUD Methoden zu implementieren anstatt für jeden Layer eine Model Klasse, eine Service Klasse und eine Repository Klasse zu erstellen
